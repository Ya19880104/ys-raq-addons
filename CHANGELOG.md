# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-03-13

### Added
- **報價歷史紀錄系統** — 自動攔截報價提交，儲存為 CPT（`ys_raq_quote`），後台可查看完整報價清單與詳情
- **報價列表頁面** — 使用 `WP_List_Table` 呈現，支援搜尋、狀態篩選、排序、分頁、批次操作
- **報價狀態管理** — 5 種狀態（新的 / 處理中 / 已回覆 / 已過期 / 已關閉），列表頁可透過下拉選單快速變更
- **自訂表單欄位** — 支援 9 種欄位類型（文字、電子郵件、電話、數字、多行文字、下拉選單、單選按鈕、核取方塊、日期），可拖曳排序、啟用/停用
- **原生欄位控制** — Name、Email、Message 欄位的顯示/隱藏與必填控制
- **電子郵件增強** — 收件人管理（主要收件人、CC、BCC）、自訂欄位資料隨附郵件
- **產品資訊顯示設定** — 控制報價頁面與郵件中顯示的產品資訊（圖片、價格、SKU、數量、小計、總計）
- **產品縮圖寬度設定** — 可自訂報價頁面產品圖片寬度（30-300px）
- **頁面按鈕控制** — 「返回商店」與「更新清單」按鈕的顯示/隱藏與文字自訂
- **電商工具箱選單** — 共用的 `ys-toolbox` 頂層選單，支援多外掛共存
- **報價頁面樣式優化** — 自訂 CSS 美化 YITH 報價頁面表格與表單

### Changed
- 後台配色方案改為淺藍莫蘭迪色系
- 管理介面全面改版，使用卡片式佈局與分頁式設定

## [1.0.0] - 2026-02-01

### Added
- **迷你詢價車 Widget** — 類似 WooCommerce Mini Cart 的下拉式詢價清單
- **即時 AJAX 更新** — 新增/移除產品時自動刷新
- **響應式設計** — 桌面版 hover 顯示，行動裝置 bottom sheet 效果
- **Shortcode 支援** — `[ys_raq_mini_cart]` 與 `[ys_raq_count]`
- **模板可覆寫** — 複製到佈景主題即可自訂外觀
